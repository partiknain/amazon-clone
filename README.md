# 🛒 Amazon Clone (HTML & CSS)

A front-end clone of the Amazon homepage built using **HTML5** and **CSS3**.
This project replicates the layout, structure, and basic styling of the original Amazon website.

---


## 📌 Features

* ✅ Amazon-like Navbar
* ✅ Search Bar UI
* ✅ Hero Section / Banner
* ✅ Product Grid Layout
* ✅ Footer Section
* ✅ Clean UI Design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
